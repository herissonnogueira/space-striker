# Space Striker

Um jogo de nave espacial desenvolvido com JavaScript vanilla.

## Sobre o Projeto

Space Striker é um jogo onde você controla uma nave espacial que deve destruir inimigos que caem do topo da tela. O projeto foi desenvolvido para demonstrar conhecimentos em JavaScript, manipulação de canvas e lógica de jogos.

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## Como Jogar

1. Abra o arquivo `index.html` no seu navegador
2. Use as setas ← → para mover a nave
3. Pressione ESPAÇO para atirar
4. Destrua os inimigos e sobreviva o máximo possível!
