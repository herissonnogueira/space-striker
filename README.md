# Space Striker

Um jogo de nave espacial arcade desenvolvido com JavaScript vanilla.

## Sobre o Projeto

Space Striker é um jogo de tiro espacial onde você controla uma nave que deve destruir ondas de inimigos que caem do topo da tela. O projeto foi desenvolvido para demonstrar conhecimentos em JavaScript, manipulação de canvas HTML5 e lógica de jogos.

## Funcionalidades

- Sistema de movimentação suave com controles de teclado
- Sistema de tiro com projéteis
- Spawn procedural de inimigos
- Sistema de colisões preciso
- Pontuação e sistema de vidas
- Dificuldade progressiva (aumenta a cada 100 pontos)
- Efeitos visuais de explosão com partículas
- High score persistente (salvo no navegador)
- Game over e sistema de restart
- Tela de fundo animada com estrelas em parallax

## Tecnologias

- HTML5 Canvas
- CSS3
- JavaScript (ES6+)

## Como Jogar

1. Abra o arquivo `index.html` no seu navegador
2. Use as setas **←** **→** para mover a nave
3. Pressione **ESPAÇO** para atirar
4. Destrua os inimigos para ganhar pontos (+10 por inimigo)
5. Evite ser atingido pelos inimigos (você tem 3 vidas)
6. A cada 100 pontos, o nível aumenta e o jogo fica mais difícil
7. Quando perder todas as vidas, pressione **R** ou **ENTER** para reiniciar

## Sistema de Pontuação

- Cada inimigo destruído: +10 pontos
- Nível atual: Score ÷ 100 + 1
- High Score é salvo automaticamente
